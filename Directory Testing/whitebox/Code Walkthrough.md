## Code Walkthrough

## Definisi
Code Walkthrough merupakan teknik review kode secara formal maupun informal yang dilakukan bersama developer untuk memahami logika kode dan mengidentifikasi potensi kesalahan program.

## Modul Yang Ditinjau

### Login
Meninjau proses validasi username dan password terhadap database.

### Data Bahan
Meninjau proses tambah, ubah, dan hapus data bahan.

### Produksi
Meninjau proses pengurangan stok bahan berdasarkan jumlah produksi.

### Pembelian
Meninjau proses penambahan stok bahan setelah transaksi pembelian.

### Laporan
Meninjau proses pengambilan data dan pembuatan laporan.

## Hasil
Struktur kode dapat dipahami dengan baik dan seluruh fungsi berjalan sesuai kebutuhan sistem.
